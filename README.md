# INAV-CrossDevice-gcs
Cross-platform Web Ground Control Station (GCS) for flight controllers Using INAV Firmware. Supports mission planning, live telemetry, USB/ESP32 connectivity, and map-based waypoint editing.

The project is built using web technologies and runs on desktop, Android, and browsers with minimal installation.

Features:

• Mission planner with interactive map
• Waypoint creation and editing
• Drag-and-drop waypoint repositioning
• Mission polyline visualization
• Live drone position tracking
• Home point detection on arm
• Live flight trail
• Telemetry display (GPS, battery, signal, mode)

Connectivity

• USB (WebSerial on desktop)
• Android USB bridge
• ESP32 WiFi bridge (WebSocket)

Platforms

• Windows (Tauri build) 
• Android (Capacitor build)
• Web browser (iOS)
